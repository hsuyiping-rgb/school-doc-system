# 📋 學校行政公文系統

> AI 輔助的學校行政公文撰稿、追蹤與雲端同步系統

以單一 HTML 檔打造、零後端依賴的輕量工具，結合 Google Gemini AI 協助學校行政人員快速產出符合公文格式的「簽」、「函／稿」、「擬辦」，並提供辦理期限追蹤與 Google Sheets 雲端備份。

**🔗 線上版（免安裝，開啟即用）**：https://hsuyiping-rgb.github.io/school-doc-system/

---

## ✨ 主要功能

| 功能 | 說明 |
|------|------|
| **AI 撰稿** | 支援三種公文類型：簽、函／稿、擬辦，依正式公文格式自動產出 |
| **函的進階設定** | 受文者層級（上級／平行／下級）自動套用對應結語與引述用語；可切換「函稿」或「函」 |
| **來文解析** | 上傳 PDF／圖片／TXT，由 Gemini Vision 自動 OCR 辨識並擷取來文機關、文號、主旨、期限 |
| **公文追蹤** | 期限倒數、緊急提醒（3 天內）、狀態管理（待辦／處理中／已結案） |
| **重複防治** | 以「機關＋主旨＋期限＋類型」判定重複，存入時自動擋下，開啟頁面自動清除既有重複 |
| **Google Sheets 同步** | 上傳、下載、狀態更新、刪除；上傳前自動比對雲端去重，多裝置共用一份資料 |
| **本機優先** | 所有資料存於瀏覽器 localStorage，離線可用，雲端僅作備份 |

---

## 🏗️ 技術架構

- **前端**：單一 `index.html`（HTML + CSS + 原生 JavaScript，無框架、無建置流程）
- **AI 模型**：Google Gemini（`gemini-2.5-flash`），由使用者自備免費 API Key
- **資料儲存**：瀏覽器 `localStorage`（本機）＋ Google Sheets（雲端，選用）
- **雲端橋接**：Google Apps Script Web App（以 GET 參數溝通，內建於設定頁可一鍵複製）
- **部署**：GitHub Pages（靜態託管）

```
使用者瀏覽器 (index.html)
   │
   ├── Gemini API ──────────► AI 產稿／來文解析
   │
   └── Apps Script Web App ──► Google Sheets（公文追蹤雲端備份）
```

## ⚙️ 初次設定

### 1. 設定 Gemini API Key（必要）
1. 前往 [Google AI Studio](https://aistudio.google.com/app/apikey) 免費申請 API Key
2. 在系統「設定」頁貼上並儲存

### 2. 填寫學校基本資料（建議）
於「設定」頁填入學校名稱、承辦人、職稱，AI 產稿時會自動套用。

### 3. 設定 Google Sheets 同步（選用）
1. 建立新試算表，命名「公文追蹤」
2. 「擴充功能 → Apps Script」，貼上系統設定頁提供的程式碼（按「📋 複製程式碼」）
3. 「部署 → 新增部署作業」→ 類型「網頁應用程式」→ 執行身分「我」、存取權「任何人」
4. 複製產生的網址，貼回系統設定頁

> **更新 Apps Script 時的注意事項**：請走「**部署 → 管理部署作業 → 編輯 ✏️ → 版本選「新增版本」**」，網址才不會變、新程式碼才會生效。若按「新增部署作業」會產生另一個新網址，舊網址仍跑舊碼。

## 📝 使用流程

1. **選擇公文類型**（簽／函／擬辦），函另可選受文者層級與產出格式
2. **輸入或上傳來文**：直接貼上內容，或上傳 PDF／圖片由 AI 解析
3. **（選填）填寫擬辦建議要點與附件**
4. 點「**✨ AI 強化撰稿**」產出公文
5. 確認後點「**💾 存入追蹤**」，自動同步至 Google Sheets
6. 於「**📊 追蹤**」頁管理期限與狀態

## 📐 公文格式規則

### 函的結語依受文者層級

| 受文者層級 | 主旨結語 | 引述用語 |
|------------|----------|----------|
| 上級機關（教育局等） | 敬請鑒核。 | 奉 |
| 平行機關（同層級） | 請查照。 | 依據 |
| 下級機關（所屬單位） | 請照辦。 | 依據 |

### AI 產稿規範
- 不使用任何 Markdown 語法（`**`、`*`、`#` 等）
- 詞語間不加空格（如「貴局」「查照」「鑒核」前後皆不加空格）
- 文字精簡、符合公文語氣，不使用口語

## ❓ 疑難排解

| 問題 | 解法 |
|------|------|
| 更新後功能異常／同步失敗 | 按 **Ctrl + Shift + R** 強制重新整理，避免瀏覽器快取舊版頁面 |
| AI 回報配額用完（quota） | 確認 API Key 免費額度，或更換其他帳號的 Key（系統內建速率限制自動重試） |
| 上傳 Sheets 失敗 | 內容過長會使網址超過上限；系統已將追蹤內容精簡為「期限＋待辦工作」並限制長度 |
| 雲端日期顯示成時間戳 | Google Sheets 會自動轉換日期；新版 Apps Script 已於寫入時設純文字、讀取時還原民國日期 |
| 資料重複 | 系統存入與上傳前皆會去重；開啟頁面也會自動清除既有重複 |
